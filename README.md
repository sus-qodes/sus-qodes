- Building **EduTube** — a distraction-free YouTube learning platform
- Exploring **GANs, LoRA, RAG, and Model Context Protocol**

### EduTube — Distraction-Free Learning

`React` `Node.js` `Tailwind CSS` `Firebase` `Gemini API`

A web platform for saving and managing YouTube playlists inside a custom embedded player — zero distractions, full focus.

- Playlist categorization and lecture scheduling
- Pomodoro timer, to-do list, and note-taking tools
- Dark theme UI with smooth animations
- Gemini AI integration for doubts, quizzes, and interactive learning

[![Live Demo](https://img.shields.io/badge/Live_Demo-7c3aed?style=for-the-badge&logo=vercel&logoColor=white)](https://edutube1404.vercel.app/)

### AI Image Detector

`Python` `PyTorch` `OpenCV`

A deep learning model for classifying real vs. AI-generated images with a hybrid architecture approach.

- Hybrid **Swin Transformer + MobileNetV3** with gated fusion
- Patch extraction and test-time augmentation (TTA)
- Evaluated on benchmark datasets for detection accuracy
- Research-grade experimentation and performance analysis
